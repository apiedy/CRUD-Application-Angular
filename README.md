# CRUD-Application-Angular
This is a basic CRUD application using AngularJS

### CRUD Application
A CRUD application is an application which does the 4 most basic operations on data being:
- C - Create
- R - Read
- U - Update
- D - Delete

### Instructions:
```
bower install --save
browser-sync start --server
```
